# Registry System (Deprecated)

> **This document describes the old registry pattern which has been replaced by explicit wiring.**
> See [Explicit Wiring](content/docs/architecture/registry.md) for the current architecture.

The old `pkg/registry/` system used `init()` functions and global registries. This has been replaced by:

- **Explicit wiring** in `pkg/app/wire.go` (generated by `forge generate`)
- **Constructor injection** with typed `Deps` structs
- **No `init()` functions** or import side effects
- **Config-driven construction** from the config proto

For the current architecture, see:
- [Architecture Guide](guides/architecture.md)
- [Getting Started](guides/getting-started.md)
